Mission statement

Provide a set of tools to manage, configure and build complex, package based, multi-variant (multi-architecture, multi-board, multi-toolchain) C/C++ projects.

xPacks overview

xPacks are general purpose software C/C++ packages, intended to enhance code sharing and reusing during the development of C/C++ libraries and applications, much the same as npm modules do so nicely in the JavaScript ecosystem.

xPack tools

The main xPack tools are:

  • xpm - the package manager
  • xmake - the package builder
  • xcdl - the component & configuration framework
